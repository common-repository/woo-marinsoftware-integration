Woocommerce Integration with MarinSoftware

== Description ==

Woocommerce Integration with MarinSoftware http://www.marinsoftware.com.

Using the plugin means that all of your product views will get inserted the proper woocommerce product id into MarinSoftware tracking script.
Order id and order total will get passed properly on MarinSoftware from order confirmation page.

== Installation ==

Installing "Woo Marinsoftware Integration" can be done either by searching for "Woo Marinsoftware Integration" via the "Plugins > Add New" screen in your WordPress dashboard, or by using the following steps:

1. Download the plugin via WordPress.org
1. Upload the ZIP file through the 'Plugins > Add New > Upload' screen in your WordPress dashboard
1. Activate the Woo Marinsoftware Integration plugin through the 'Plugins' menu in WordPress
